## Configuration Reference

This is the configuration reference details for this action. You can setup and configure this action through your Traffic Policy documents.

### Action Type

`compress-response`

### Configuration Fields

| Parameter    | &nbsp;              | Description                                                                                                                                       |
| ------------ | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| `algorithms` | `[]string` | The list of allowed compression algorithms to be considered during encoding type negotiation. Optional. Defaults to `gzip`, `deflate`, `br`, `compress`. |

### Supported Compression Algorithms

Full list of supported compression algorithms:

| Algorithm  |
| ---------- |
| `br`       |
| `compress` |
| `deflate`  |
| `gzip`     |

### Supported Directions

- `inbound`
- `outbound`

### Supported Schemes

- `https`
- `http`
