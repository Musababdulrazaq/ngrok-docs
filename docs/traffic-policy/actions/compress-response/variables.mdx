## Result Action Variables

The following variables are made available for use in subsequent expressions and
string interpolations after the action has ran. Variable values will only apply
to the last action execution, results are not concatenated.

| Name                                          | Type    | Description                           |
| --------------------------------------------- | ------- | ------------------------------------- |
| `actions.ngrok.compress.already_compressed`   | `boolean` | Will be `true` if the body was already encoded. |
| `actions.ngrok.compress.negotiated_algorithm` | `string`  | The algorithm selected by the action. |
